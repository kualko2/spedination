## What Spedination does

Spedination organizes full truck load (FTL) transports across Europe.

Key principle:
- one truck
- no transshipment
- one contact person

Focus:
- reliable execution
- clear communication
- fast problem solving

This is not a generic logistics company.  
It is a hands-on, digital freight setup with a strong operational focus.

## Writing Style (Spedination Tone)

All written content must follow this style.

### Core principles
- Write direct and clear
- Keep sentences short
- Focus on real problems
- Avoid complexity
- Sound human

### Tone
- confident
- practical
- slightly provocative

### Avoid
- buzzwords
- corporate language
- empty claims

### Write like this
- real situations
- simple language
- clear message

### Spedination writing pattern

1. Describe a real situation  
2. Show what goes wrong  
3. Show simple solution  
4. Show difference  
5. Ask simple question  

### Example (correct)

Most transports don’t fail because of the route.  
They fail because too many people are involved.

We keep it simple:  
one truck, one contact, no transshipment.

### Example (wrong)

We provide innovative logistics solutions tailored to your needs.
